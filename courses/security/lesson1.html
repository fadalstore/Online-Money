Ku soo dhowow Koorsada Cybersecurity!

Casharkaan wuxuu kugu barin doonaa waa maxay Ethical Hacking, farqiga u dhexeeya hacker-ka ansaxa ah iyo kan xunka, iyo sababta cybersecurity ay aad ugu muhiim tahay adduunka maanta.

📌 Waxaad baranaysaa: Macnaha cybersecurity, noocyada hackers, hawsha pen tester, framework-ka ethical hacking, iyo sharciga kugu xidha.

⚖️ Ogeysiis Qanoon ah: Aqoonta ku isticmaal si ansax ah oo sharciga la waafaqsan. Isku day weerar nidaam aad fasax u hayso. Xukumaddu waxay u aragtaa hacking sharci darrada ah inay tahay dambiyada ugu culus.

1. Waa maxay Cybersecurity?

Cybersecurity waa xirfadda iyo nidaamka ilaalinta nidaamyada kompyuutarka, shabakadaha, iyo xogta ka dhan ciidamada aan la ogolayn. Waxay ka kooban tahay:

  • Network Security: Ilaalinta shabakadaha internetka
  • Application Security: Ilaalinta software iyo apps-ka
  • Information Security: Ilaalinta xogta khaasada
  • Operational Security: Nidaamyada iyo habsami-socodka ilaalinta
  • Disaster Recovery: Sida loo soo celiyo nidaamka weerarka ka dib

Bug Bounty Hunters

Nooc kale oo muhiim ah: Bug Bounty Hunters. Waxay baaraan shabakadaha shirkadaha weyn (sida Google, Facebook, iyo Microsoft) si ay u helaan nuglaanshaha, oo shirkaddu lacag u bixiso marka la helayo. Kuwaas waa shaqo lacag leh oo halal ah!

3. Ethical Hacking — Hawsha Pen Tester

Penetration Tester (Pen Tester) waa xirfadle la siiyay fasax in uu weeraro nidaam si loo ogaado nuglaanshaheeda. Wuxuu fuliyaa:

  1. Qorsheysiinta (Planning): Heshiis sharci ah iyo go'aanka wixii la doorbidayo
  2. Ogaanshaha (Reconnaissance): Soo ururinta macluumaadka nidaamka
  3. Baadista (Scanning): Raadinta meelaha furan iyo nuglaanshaha
  4. Helista Galitaanka (Exploitation): Isku dayo gelitaanka si loo xaqiijiyo nuglaanshaha
  5. Xogtaynta (Post-Exploitation): Waxii la geli karo iyo waxa la geli kari waayay
  6. Warbixinta (Reporting): Daabacaad buuxda oo leh xalal

4. CIA Triad — Aasaaska Amniga

Cybersecurity-ga waxaa lagu qiyaasaa saddex mabaadi'o oo aasaasi ah:

🔒 C — Confidentiality (Sirta)

Xogtu waa inay marin hawleed u noqotaa kuwa ansax ah keliya. Haddaan cidna la ansixin, ma geli karaan.

✅ I — Integrity (Saxnaanta)

Xogtu waa inay ahaataa mid saxan oo aan la beddelinnin cidna oo aan la ansixin.

📡 A — Availability (Helitaanka)

Nidaamyada iyo xogtu waa inay u diyaar yihiin kuwa fasax leh marka ay u baahan yihiin.

5. Shaqooyinka Cybersecurity — Fursadaha

Cybersecurity waa mid ka mid ah meelaha ugu baahida badan shaqaalaha adduunka. Waxaa jira fursado badan:

  • 💼 Penetration Tester: $80,000–$150,000/sano (Maraykan)
  • 🛡️ Security Analyst: $60,000–$120,000/sano
  • 🔐 Security Engineer: $100,000–$180,000/sano
  • 🏆 CISO (Chief Info Security Officer): $150,000–$300,000+/sano
  • 🐛 Bug Bounty Hunter: $500–$100,000+ (mid-mid)
  • 🌍 Freelance Security Consultant: $50–$200+/saac

6. Shahaadadaha Cybersecurity

Haddaad doonayso inaad xirfadle cybersecurity ah noqotid, waxaan kugula talineynaa:

  • CompTIA Security+ — Bilow wanaagsan ee xirfadlayaasha cusub
  • CEH (Certified Ethical Hacker) — Mid ka mid ah kuwa ugu caan badan
  • OSCP (Offensive Security Certified Professional) — Ugu adag, laakiin ugu qiimaha badan
  • eJPT (eLearnSecurity Junior Pen Tester) — Mid fudud oo bilowayaasha ku habboon

7. Agabyada (Tools) Ugu Caansan

  • 🗺️ Nmap: Port scanner — ogaanshaha adeegyada furan (Casharka 5aad)
  • 🌐 Wireshark: Network packet analyzer
  • 💥 Metasploit: Framework-ka exploitation-ka
  • 🕷️ Burp Suite: Web application security testing
  • 🔑 John the Ripper: Password cracking
  • 📡 Aircrack-ng: WiFi security testing
