📘 Heerka 1aad: Aasaaska Amniga (Foundations)
1
Waa maxay Ethical Hacking? — Hordhaca Amniga
Faahfaahinta Ethical Hacking, farqiga u dhexeeya "white hat" iyo "black hat", iyo sharciga cybersecurity. ~45 daqiiqo.
→
2
Networking Basics — Sida IP Addresses iyo Ports u Shaqeeyaan
Aasaaska shabakadaha: IP addresses, ports, protocols (TCP/UDP), iyo sida xogtu u safarto internet-ka. ~50 daqiiqo.
→
3
Linux for Hackers — Maxay Muhiim u Tahay?
Sababta Linux ay u tahay nidaamka ugu caansan cybersecurity, amarka aasaasiga ah, iyo sida loo dejisto Kali Linux. ~55 daqiiqo.
→
⚔️ Heerka 2aad: Tools & Vulnerabilities
4
Information Gathering — Footprinting & OSINT
Sida loo baaro mareegaha iyo networks-ka si lagama-maarmaan u isticmaalka amniga: WHOIS, DNS lookup, iyo Google Dorking. ~55 daqiiqo.
→
5
Scanning & Enumeration — Isticmaalka Nmap si Xirfad leh
Nmap — tool-ka ugu caansan ee port scanning: sida loo baaro nidaamyada, helo adeegyada furan, iyo ogaado nidaamka hawlgalka. ~65 daqiiqo.
→
6
Web Security — Vulnerabilities & Directory Discovery
Sida loo aqoonsado nuglaanshaha shabakadda: XSS, SQL Injection (hordhac), directory discovery, iyo sida website-ka loo ammaan geliyo. ~70 daqiiqo.
→
